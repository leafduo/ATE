ATE ── Are They Equivalent? 用来判断两个命题逻辑的公式是否等值，免去你画真值表、等值演算、用归结法的时间，你只需要输入两个公式，计算机就会告诉你，它们是否等值。

使用方法是运行 ate（或 ate.exe），根据提示输入两个公式，计算机会告诉你它们是否等值；或者把两个公式当作命令行参数传入 ate（或 ate.exe），也可以得到结果。（使用第二种方法时，建议使用引号把表达式引起来）

公式中使用单个英文字母代表变元，支持圆括号，支持合取（&）、析取（|）、蕴含（->）、否定（!）、异或（^）等操作符。

原理是用真值表的方法来判断公式是否等值，对公式求值的方法是采用栈的方法来进行表达式求值，判断是否所有赋值后的公式都等值，如果是，则原公式等值。

在以下平台上测试通过：
Mac OS X 10.6 (Leopard) GCC 4.2.1 GNU Make 3.81
Debian GNU/Linux 4.0 kernel 2.6.31.5 GCC 4.1.2 GNU Make 3.81
Windows Vista MinGW GCC 3.4.2 GNU Make 3.80
